# Ganttopia

Ganttopiaは、FastAPIとVanilla JavaScript（HTML/CSS）で構築された、直感的に操作できるローカル向けのガントチャート・デスクトップアプリケーションです。`pywebview`を使用しており、ブラウザを開くことなく独立したウィンドウで動作します。

## 主な機能

*   **直感的なドラッグ＆ドロップ操作:**
    *   チャート上のバーの真ん中を掴んでドラッグし、日程を前後に移動。
    *   バーの左右の端を引っ張って、期間（日数）を伸縮。
    *   左側のリストで、タスクを上下にドラッグして順番を入れ替え。
*   **複数ページ（タブ）管理:**
    *   チャートを複数ページに分けて管理可能。タブの追加、名前変更、削除が簡単に行えます。
*   **工数管理:**
    *   タスクごとに予定工数（時間）を設定し、ページ全体の合計工数を自動集計。
*   **サブタスク（階層）管理:**
    *   親タスクを指定してサブ項目を作成し、ツリー状に折りたたみ・展開が可能。
*   **カスタマイズ可能なデザイン:**
    *   タスクごとに10色のカラーパレットからお好みのテーマカラーを設定可能。
    *   日・週・月へのスケール切り替え機能。
*   **エクセル出力:**
    *   タスクの一覧表や、ガントチャートの見た目をそのままExcelファイル（.xlsx）としてデスクトップにワンクリックで出力。

## 技術スタック

*   **バックエンド:** Python, FastAPI, SQLite, SQLAlchemy
*   **フロントエンド:** HTML5, Vanilla JavaScript, CSS3 (外部ライブラリ非依存の軽量設計)
*   **デスクトップ化:** pywebview
*   **エクセル出力:** openpyxl

## 起動方法（開発環境）

本プロジェクトはパッケージマネージャに `uv` を使用しています。

### 1. 依存関係のインストール

```bash
uv sync
```
*(または `uv pip install -e .` など、お使いの環境に合わせてインストールしてください)*

### 2. アプリケーションの起動

以下のコマンドを実行すると、ローカルサーバーが立ち上がり、自動的にデスクトップウィンドウが開きます。

```bash
uv run python main.py
```

## EXE化（配布用ファイルの作成）

Windows向けにスタンドアロンの実行ファイル（.exe）を作成することができます。

### 1. ビルドの実行

プロジェクトのルートディレクトリにある `build.bat` をダブルクリックして実行するか、コマンドプロンプトで以下を実行します。

```bash
build.bat
```

### 2. 生成物の確認

ビルドが完了すると、プロジェクト内に `dist` フォルダが作成され、その中に `Ganttopia` という**フォルダ**が生成されます。
そのフォルダ内にある `Ganttopia.exe` をダブルクリックすると、一瞬でアプリが起動します。

> [!TIP]
> 他の人に配布する場合は、この `Ganttopia` フォルダを**フォルダごとZIPファイル等に圧縮**して配布してください。フォルダ内の他のファイル（DLLや設定ファイル）が欠けると起動しなくなります。

## データの保存について

入力されたタスクデータはすべてローカルの `gantt_data.db` (SQLiteデータベース) に保存されます。
クラウド等へのデータ送信は一切行われないため、機密性の高いプロジェクトでも安全にご利用いただけます。
（※ `gantt_data.db` は `.gitignore` に指定されており、Gitにはプッシュされません）
